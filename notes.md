# Notes

## Prototypes

Twee hoofdbestaansrechten van prototypes:
- Overerving
  ```js
  class A extends B {}
  class Cat extends Animal {}
  ```
- Extension methods

Prototypes in JS

- hele funtionele overerving
- dynamischer/flexibeler dan C#/Java
  - @ runtime kun je het prototype gewoon aanpassen:
    ```js
    Object.setPrototypeOf(obj, { x: 24 });
    ```
